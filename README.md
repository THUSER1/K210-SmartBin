# Smart Waste Sorting System

**An AI-powered smart waste sorting system** combining visual recognition, voice interaction, robotic arm handling, and Android IoT integration.  
Built with **K210 (AI vision)**, **STM32 (control & voice)**, and an **Android mobile app**.

## Features
- **AI Garbage Recognition (K210)** – Fast offline CNN-based waste classification  
- **Voice Interaction** – Supports voice commands and voice feedback  
- **Robotic Arm Control (STM32)** – Automatic grasping and sorting  
- **Android IoT App** – Real-time device status and data display
